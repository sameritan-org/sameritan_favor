# Sameritan Favor Node

Decentralized storage node for the Sameritan network.

## Overview

Favor Node enables members to provide storage and bandwidth to the Sameritan network in exchange for Favor coins. Run a node to earn Favor while supporting the decentralized infrastructure.

- [Documentation](https://sameritan.org/docs/coins/favor)
- [Developer Resources](https://sameritan.org/developers)
